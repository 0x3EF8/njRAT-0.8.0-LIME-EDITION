Welcome to njRAT Lime Edition Support

Reading this file will help you to fix any problem

If this your first time using Lime Edition, I recommend that you use |NYAN| as your key, as it the default. Also, port 6522 is the default for Lime Edition

If you for some reason decided to use another Key name, Please watch the setup video that is included.

After you create your stub, you must use it on your local machine, download and install VMware and then test the connection. use port 127.0.0.1 for the test.

So now I will assume that your setup is good, let's see what issue you may face.

[1] I ran the stub but can't see myself as client
# It means that you have a wrong port or wrong key or the wrong IP. read the paragraph above.

[2] I see client, But remote desktop and file manager and cam are not working
# That means you haven't seen the setup video, please watch it.

[3] I see that I have connections but I have no visible client
# This is tricky, This means you are using a port that you have already use it. let say you have 5 clients on DarkComet on port 5566, Then you open njrat on same port which is 5566. so these clients are not using lime edition stub so that's why there is a connection on port 5566 but it will never be visible. That's why I recommend you use port 6522 as it lime's default.




Let's talk about functions:

Bitcoin Grabber
Will replace your address instead of the receiver address on everytime the client open online wallet or local software wallet, if he trying to send BTC to somebody, your address will be always as the receiver.

Ransomware
Encrypt clients files. it's very recommended to use it on a client that has i7 or i5

Screen Locker
It will lock the client screen for good. ALT+F4 and windows key both will be disabled. If the client tries to restart the PC, It will be locked as he login again. The only way to unlock it is to press UNLOCK. It's also recommended to use anti-process as a combo.

Malware Killer
Known as Bot Killer, It will kill any RAT on your clients PC. BUT if you want to use it, make sure when you use crypter to choose "inject itself". Using regasm or whatever will kill your client for good.

Anti process
It will kill Client.exe when the user is using Vmware or Sandboxie to avoid AV companies or debuggers. But it will kill any analysis software such as process hacker or windows task manager.

Delete Logs
This will delete any fingerprints or logs from windows OS

Timeout
is time to reconnect again after your close Lime edition, I recommend to use 5 sec.
